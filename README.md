# 🔐 Cyber Security Write Ups
### Author: [Gerónimo Vera](https://github.com/gero04)

Welcome to my personal logbook! 🧠

This repository contains my detailed write-ups for cybersecurity challenges across multiple platforms. The purpose of this space is twofold: to document my own learning process and the techniques I am acquiring, and to share this knowledge with the community so that others can learn and replicate the steps.

Here you will find not only the commands used, but also the methodology, the tools, and most importantly, the **way of thinking** (the "mindset") behind each vulnerability.

---

## 📁 Structure

Each platform has its own directory. Within each one, you will find folders named after each category (or the lab name if no categories exist). Inside, a `README.md` with the complete write-up, and in some cases, scripts or support files.
```
CyberSecurityWriteUps/
├── RootMe/
│   └── WebClient/
│       └── HTML - Disabled Buttons/
│           └── README.md
└── SoftwareSeguro/
    └── ...
```
